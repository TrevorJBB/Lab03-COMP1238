# Number Row Characters

- `!` - Exclamation mark
- `@` - At symbol
- `#` - Hash, Pound, Number sign
- `$` - Dollar sign
- `%` - Percent sign
- `^` - Caret, Circumflex
- `&` - Ampersand
- `*` - Asterisk
- `(` - Open parenthesis
- `)` - Close parenthesis
- `_` - Underscore
- `+` - Plus sign
