# Number Row Characters

- `$` - Dollar sign
- `%` - Percent sign
- `^` - Caret, Circumflex
- `&` - Ampersand
- `*` - Asterisk
- `(` - Open parenthesis
- `)` - Close parenthesis
- `_` - Underscore
